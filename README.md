# SIMCLO
simclo is a simple digital clock for your terminal.
No config files, no options, just time.

![preview](https://github.com/Antoniowski/simclo/blob/main/simclo_preview.png?raw=true)

## How to install
Clone the repo, enter in the root directory and run the <code>install.sh</code> script contained in the repo:<br/>
```
./install.sh
```
The script will compile the program and will install it on your system
